## Notes

Started learning Rust today! Went through the first few chapters of "The Rust Programming Language" book.

Key concepts learned:
- Ownership and borrowing - this is the big one. Every value has a single owner.
- The borrow checker prevents data races at compile time
- `&` for references, `&mut` for mutable references
- Only one mutable reference OR multiple immutable references (not both)

Example that helped it click:
```rust
let s1 = String::from("hello");
let s2 = &s1;  // s2 borrows s1
println!("{}", s1);  // still works because s2 is just borrowing
```
